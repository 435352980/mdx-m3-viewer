Class: ModelViewer

ModelViewer

The main model viewer object.

new ModelViewer(canvas, urls, onmessage, debugMode)

Parameters:
Name Type Description
canvas HTMLCanvasElement A canvas element.
urls object An object with the necessary methods to get urls from the viewer.
onmessage function A callback function, which the viewer will call with messages.
debugMode boolean If true, the viewer will log the loaded models and their parser to the console.

setShader(id)

Sets the shader. Possible values are 0 for `standard`, 1 for `diffuse`, 2 for `normals`, 3 for `uvs`, 4 for `normal map`, 5 for `specular map`, 6 for `specular map + normal map`, 7 for `emissive`, 8 for `unshaded`, 9 for `unshaded + normal map`, 10 for `decal`, and finally 11 for `white`. Note: only the normals, uvs, and white shaders affect Warcraft 3 models, the rest only affect Starcraft 2 models.
